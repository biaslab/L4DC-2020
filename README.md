## Proof of concept for online hybrid message passing inference for AR-HGF.
To run the experiments from AR-HGF.ipynb you need to install the following dependencies:
- ForneyLab.jl
- PolynomialRoots.jl
- Distributions.jl
- FastGaussQuadrature.jl
- ForwardDiff.jl
- ProgressMeter.jl
- LaTeXStrings.jl
- DSP.jl
